# Bảng phân bổ Passage Vocabulary theo Cert Level

Hệ thống lọc bài đọc (Passage) dựa trên độ khó (level) của từng bài so với cấp độ (cert) mà học sinh đang chọn:
* Học sinh ở cấp độ nào sẽ được xem các bài đọc có độ khó `level <= cert` đó.

- **Starters (Level 1):** Chỉ thấy bài `starters`
- **Movers (Level 2):** Thấy bài `starters` + `movers`
- **Flyers (Level 3):** Thấy tất cả các bài `starters` + `movers` + `flyers`
